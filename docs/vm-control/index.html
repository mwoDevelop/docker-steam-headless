GitHub Pages + Cloud Run

VM Control

Secure control panel for a single GCE VM. The page signs in with Google, then calls a Cloud Run API that starts, stops, restarts, or inspects the instance directly.

No GitHub PAT is required in this mode. The page stores only the backend URL and a short-lived Google session token in the browser session.
